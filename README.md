# Trybe

This repository contains all the activities of learning developed by Paulo Porto while studying on [Trybe](https://www.betrybe.com/) :rocket:

_"Trybe is a school of future to anyone who want improve in life and build a successful career in technology, where the person only pay when get a good job."_

The course has more than 1,500 hours of classroom and online classes, starting from: Introduction to software development, passing throught front-end, back-end, computing science, software engineering, agile methodologies and soft skills.
